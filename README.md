# Practica de Colaboración con Ramas

Este repositorio contiene la práctica guiada de Git y GitHub para aprender a trabajar en equipo utilizando ramas, colaboradores, *issues* y *pull requests*.

## 📝 Descripción del Proyecto
El objetivo principal es simular un entorno de trabajo real construyendo una clase `Calculadora` en Java. A lo largo del proyecto, se realizarán diferentes integraciones de código.

## 🚀 Fases de la Práctica
* **Parte 1 y 2:** Creación del repositorio y configuración de colaboradores.
* **Parte 3:** Creación del código base de la `Calculadora`.
* **Parte 4 y 5:** Trabajo colaborativo simulado integrando una rama externa mediante un *Pull Request* (PR).
* **Parte 7 y 8:** Creación de un *Issue* para registrar una nueva tarea y resolución del mismo utilizando una rama de características (`feature-restar`).

## 🛠️ Tecnologías
* Java
* Git & GitHub
